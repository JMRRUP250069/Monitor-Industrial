# Monitor Predictivo Industrial 
Prototipo de análisis de datos térmicos para la prevención de fallas en brazos robóticos industriales.

## Características
- **Análisis Predictivo:** Cálculo de tendencia lineal para estimar tiempo de fallo.
- **Trazabilidad:** Registro automático de eventos y errores con marca de tiempo.
- **Filtro de Integridad:** Validación de datos de sensores para evitar falsos positivos.
- **Interfaz Moderna:** Desarrollado con Python y CustomTkinter (Dark Mode).

## Stack Tecnológico
- Python 3.x
- CustomTkinter (UI/UX)
- Analítica de datos básica
